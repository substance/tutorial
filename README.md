# How to update the tutorial

## Checkout reader

    $ git clone https://github.com/substance/reader substance-reader
    $ cd substance-reader

## Manage dependencies and assets
  
    - Adjust `project.json`
    - Brute force! copy style from io `styles/substance.css`

## Bundle reader
  
    $ substance --update
